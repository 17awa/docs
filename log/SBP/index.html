# SBP基金会

SBP基金会全称特殊收容措施基金会，是以一个怪诞科幻类的架空世界为主题的网络共笔文学小说集网站，同时也是及其衍生相关作品中的神秘虚构组织，专门发现各种超自然现象、神秘生物等异常事件。

我们的运作不受任何国家组织干涉管辖，目标为收容世界上的异常现象、事件、个体等等，并统称为“收容物”“（即SBP）”。每个收容物都会有一个对应的编号（如SBP–XXXX）。许多这些“物品”既可对人产生物理危险，又可对世界范围的人群产生心理影响，动摇他们的个人信念，干扰他们的日常生活。基金会则致力于控制这些异常并将其收容，在不为人知的情况下保护着这个世界。

人员表

S级人员:SBP基金会的最高层,负责对SBP的各种管理和掌控SBP的机密资料
A级人员:掌握了一些机密资料负责巡查SBP
B级人员:观察SBP和进行一些实验,但是被A级,S级人员管理着
C级人员:重刑犯,进入危险的SBP(WRARING)的收容间进行探索
D级人员：死刑犯(如:搞家好,李加姨),对各种等级的SBP进行探索
(ZAYIN,TETH,HE,WAW,ALEPH)
记住,D级人员的死对宇宙,世界,社会没有影响!
